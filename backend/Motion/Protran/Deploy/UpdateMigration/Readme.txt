This directory contains all files needed for an update migration from a customer system using Infomedia 1.0.x or TFTPlayer and TFTUpdate to TFTStartup.bat and install Gorba’s current software like Protran, Infomedia, MediServer etc. via USB.

See also TD_UpdateMigration.docx for more information.

The following folders exist:
- "Batch File to Convert": the batch file that needs to be converted to an EXE using "Advanced BAT to EXE Converter"
- "ImageUpdate": an example for image update, make sure to actually use the image update configured for your project
- "Program_Update": the folder to be copied into the root of the USB stick (replace "Infomedia.exe" with the file created with "Advanced BAT to EXE Converter")
