Prerequisites:
- BackgroundSystem project requires Asp.net MVC 3
- Microsoft Code Contracts (http://msdn.microsoft.com/en-us/devlabs/dd491992)
- When run on TFS, the configuration platform must be ".NET" or "Mixed platforms"
- The DataImporter needs the OpenXml SDK 2.0

- Some PowerShell script require PowerShell Community Extensions (http://pscx.codeplex.com/)

Warnings:

- To run the services which interact with Sql Server the Distrubute Transaction Coordinator service must be started
